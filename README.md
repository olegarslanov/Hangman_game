# Hangman_game

This is program for play "Hangman game"
 
Player have 10 lifes 
    * maximum quess attempts: 10
    * ability to quess a word or a letter. if a quess is incorrect user loses 1 life
    * if user has 0 quesses (lifes) game is end

How to start:

Run in terminal windows commands:
1. git clone https://github.com/olegarslanov/Hangman_game
2. pip install -r requirements.txt
3. run py main.py from src folder
*. requires Python 3.10.x - recommended version (but working with 3.14.x version too) !!!
